Le logiciel est obligatoirement à ouvrir avec le logiciel de programmation "EduPython"

AVANT DE JOUER AU JEU :
- Installez le logiciel "EduPython".
- Installez les modules "pytmx", "pyscroll" et "dataclasses" dans votre logiciel "EduPython". 
>>> Pour cela, lancez "EduPython", allez dans "Outils" dans le menu supérieur puis dans "Outils" enfin, cliquez sur "installer un module avec pip". 
Mettez "pytmx" dans la zone de texte, cliquez sur "OK" et refaite la même manipulation pour "pyscroll" et "dataclasses".
- Si logiciel et module installez, vous pouvez cliquez sur la fléche verte sur la barre supérieur dans "EduPython".

REGLE :
- Faire une tour de circuit le plus rapidement possible
- Si vous touchez l'herbe, vous serez ralenti
- Si vous touchez le gravier, vous serez grandement ralenti et arrêtez

COMMANDE :
- Flèche directionnelle pour se déplacer. 
|- Haut pour aller en haut
|- Bas pour aller en bas
|- Droite pour aller à droite
|- Gauche pour aller à gauche

DROIT DE MUSIQUE
- Musique exclusive au jeu.
